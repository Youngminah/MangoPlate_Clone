# 🥭 망고 플레이트 클론 2주 프로젝트

망고플레이트는 iOS 앱스토어에 실제 존재하는 맛집 조회 어플입니다. 

2주동안 망고플레이트 어플을 우선순위를 정하여 그에 따라 최대한 똑같이 클론 코딩하여 앱을 구성하였습니다.
<br></br>
### ▶︎ 협업자
클라이언트 1명, 서버 1명.

<br></br>
### ▶︎ 망고플레이트 클론 기능

| 기능                    | 진행      | 코멘트                             |
| ---------------------- | -------- | -------------------------------- |
| 소셜 로그인(카카오) | ✔️        |   서버에서 JWT토큰을 받아 자동로그인 구현 완료.                               |
| 맛집 평점,조회,거리,리뷰순을 정렬 선택 가능  | ✔️       |                           |
| 원하는 맛집 필터 카테고리 선택 가능   | ✔️       | 중복 선택 가능 |
| 광고 자동 슬라이딩   | ✔️       |  |
| 리뷰 조회  | ✔️       | 맛있다, 괜찮다, 별로 필터로 선택 조회 가능 |
| 리뷰 쓰기  | ✔️       | 사진 여러개 첨부 가능 (파이어베이스 이용)|
| 리뷰 수정  | ✔️       |  |
| 리뷰 삭제  | ✔️       |  |
| 디테일한 애니메이션  | ✔️       |  |
